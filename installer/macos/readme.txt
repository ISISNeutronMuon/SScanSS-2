Build Mac OS Disk Image
-----------------------
The sscanss 2 disk image uses the create-dmg tool to build. This is installed by running the following command.

   > brew install create-dmg

Once create-dmg is installed, The sscanss 2 disk image is built using the following commands.
   
   > chmod +x build_dmg.sh (run once)
   > ./build_dmg.sh
